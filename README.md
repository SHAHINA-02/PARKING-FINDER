## ABOUT

DubaiPark is a parking discovery and booking platform built specifically for Dubai. Users can search for available parking spots by location, mall, or street name — and filter by neighbourhood across Dubai's key districts. The platform combines a search interface with an interactive map view and a booking management system.

---

## HOW IT WORKS

```
Search by location / mall / street
        ↓
Filter by Area (Downtown, Marina, DIFC, JBR, etc.)
        ↓
Available Spots List  +  Interactive Map View
        ↓
Select Spot → Book → My Bookings
```

---

## FEATURES

**Parking Search**
- Search by location, mall, or street name
- Filter by Dubai area: All Areas, Downtown, Al Barsha, Dubai Marina, Business Bay, Jumeirah Beach Residence, Al Wasl, DIFC, Jumeirah 3, Deira, Dubailand
- Real-time available spots display
- Interactive map view alongside listings

**Booking System**
- My Bookings section to manage reservations
- Sign In for user account access

**UI & Design**
- Black and gold premium colour scheme
- Clean split-view: listings left, map right
- Mobile-responsive layout

## INSTALLATION

**Prerequisites:** Node.js 18+

```bash
git clone https://github.com/YOUR_USERNAME/parking-finder.git
cd parking-finder
npm install
npm run dev
```

Open `http://localhost:3000` in your browser.
